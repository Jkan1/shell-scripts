* Terminal

    - ~ (Home Directory of <username>)

* Important Commands
    - <command> --help (Shows Command Options and Usage)
    - man <command>   (Manual Page from Linux)
    - pwd (Print Working Directory)
    - uptime  (System uptime and Load, Users logged In )
